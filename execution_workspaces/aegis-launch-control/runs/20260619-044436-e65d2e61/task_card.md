# Build a polished full-stack web app called “Aegis Launch Con

Build a polished full-stack web app called “Aegis Launch Control”.

It should be a cinematic mission-control dashboard for managing the launch of an AI product. The app should feel like a modern command center: dark interface, neon accents, clear data visualization, responsive layout, and highly polished presentation.

Core requirements:

- Use Vite + React + TypeScript for the frontend.
- Add a lightweight local backend API if appropriate.
- Seed realistic launch data in the app/backend.
- Create a visually impressive dashboard with multiple sections:
  - Mission Overview with launch countdown, readiness score, blockers, velocity, deployment status, and risk level.
  - Workstream Board for Strategy, Product, Engineering, Design, Launch, and Growth tasks.
  - Dependency Map showing how key tasks depend on each other.
  - Risk Radar or risk matrix showing technical, UX, market, timeline, and resource risk.
  - Launch Timeline showing milestones from validation to public launch.
  - Scenario Simulator where the user can apply launch scenarios and see readiness/risk/timeline impact change.
- Add interactive filtering by status, priority, and workstream.
- Make the UI responsive and visually polished enough for a public demo video.
- Avoid external paid APIs or database setup.
- Include meaningful sample data and empty/error states where useful.
- Ensure the project builds successfully and can be browser-previewed.
